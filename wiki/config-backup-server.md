# 备份服务器配置 (Backup Server Configuration)

PT-depiler 支持多种备份服务，可以将配置、Cookie 等数据同步到云端。

## 🎯 支持的备份服务

- **CookieCloud**
- **DropBox**
- **Gist**
- **GoogleDrive**
- **OWSS**
- **WebDAV**

## 📋 配置说明

### WebDAV

WebDAV 是一个开放的网络协议，许多网盘都支持。

**支持的服务：**
- 坚果云
- Nextcloud
- ownCloud
- 群晖 NAS WebDAV

### 其他服务

每个备份服务都有其特定的配置要求，请参考相应服务的文档。

## ⚠️ 注意事项

- 备份数据包含敏感信息，请确保使用安全的服务
- 建议使用端对端加密的备份服务
